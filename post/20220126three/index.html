源码版本: three.js v0.135.0

如何查看源码

创建一个空白npm项目，在项目中执行npm install three之后，在项目中的node_modules/three目录中就是three.js的代码。

build目录：打包后的three.js代码。其中：

  • three.js：three.js的JS脚本版本
  • three.module.js：three.js的ES Module版本

src目录：three.js源码

建议先通过API文档熟悉类的基本功能和用法，然后再去阅读类的源码。

class Matrix3 {
    constructor() {
        this.elements = [
            1, 0, 0,
            0, 1, 0,
            0, 0, 1
        ];
    }

    // 当前矩阵 = 当前矩阵 乘以 matrix
    multiply(matrix) {...}
    // 当前矩阵 = matrix 乘以 当前矩阵
    premultiply(matrix) {...}
    // 当前矩阵 = 当前矩阵的逆矩阵
    invert(){...}
    // 当前矩阵 = 4维矩阵matrix4左上角3阶矩阵的逆转置矩阵
    getNormalMatrix(matrix4){}
    // 当前矩阵 = 当前矩阵的单位矩阵
    identity(){...}
    // 当前矩阵 = 当前矩阵的转置矩阵
    transpose(){...}
}

注意行优先列优先的顺序：

set()方法参数采用行优先(row-major)， 而它们在内部是用列优先(column-major)顺序存储在数组当中，这意味着：

1
2
3

m.set( 11, 12, 13,
       21, 22, 23,
       31, 32, 33 );

元素数组elements将存储为:

1
2
3

m.elements = [ 11, 21, 31,
              12, 22, 32,
              13, 23, 33 ];

在内部，所有的计算都是使用列优先顺序进行的。

Matrix4

4维矩阵。4x4矩阵最常见的用法是作为一个变换矩阵。

任何3D物体(Object3D)都有三个关联的矩阵：

  • Object3D.matrix：物体的局部变换矩阵(也称为本地变换矩阵。变换矩阵，也就是模型矩阵)。这是物体相对于其父物体的变换矩阵。

  • Object3D.matrixWorld：物体的全局变换矩阵(或称世界变换矩阵)。如果物体没有父物体，那么就与其局部变换矩阵matrix相同。(个人理解：比如物体A中有个物体B，物体A没有父物体。物体A的matrix是向左平移30m，而物体B的matrix是向左平移10m。那么物体A的matrixWorld就是向左平移30m，而物体B的matrixWorld就是向左平移40m)

  • Object3D.modelViewMatrix：表示物体相对于摄像机坐标系的变换矩阵(换一种说法就是，通过这个矩阵，可以将物体变换到摄像机坐标系中，这和WebGL中对modelViewMatrix的定义相同)。一个物体的modelViewMatrix等于 摄像机的世界变换矩阵的逆矩阵 乘以 物体的世界变换矩阵。(个人理解：因为物体和摄像机的相对关系，摄像机的matrixWorld的逆矩阵应该就等价于物体的viewMatrix，而物体的matrixWorld实际上就是物体的modelMatrix，因此物体的modelViewMatrix就等于二者相乘)

摄像机Camera是Object3D的子类，除了上面几个矩阵之外，它有几个额外的四维矩阵：

  • Camera.projectionMatrix: 投影矩阵。
  • Camera.matrixWorldInverse: 视矩阵(viewMatrix)，也就是Camara的matrixWorld的逆矩阵。

回顾一下WebGL体系的三大矩阵：物体的matrixWorld就是modelMatrix，Camera.matrixWorldInverse就是viewMatrix，而Camera.projectionMatrix就是projectionMatrix。

有多种选项可用于从 Matrix4 中提取位置(平移)、旋转和缩放。

  • Vector3.setFromMatrixPosition：提取位置相关的分量。
  • Vector3.setFromMatrixScale：提取缩放相关的分量。
  • Quaternion.setFromRotationMatrix，Euler.setFromRotationMatrix或extractRotation：从纯(未缩放)矩阵中提取旋转相关分量。
  • decompose：可用于一次性提取位置、旋转和缩放分量。

let _object3DId = 0;

class Object3D extends EventDispatcher {
    constructor() {
        super();

        Object.defineProperty(this, 'id', { value: _object3DId++ });

        this.type = 'Object3D';

        this.parent = null;
        this.children = []; // 一个Object3D中可以包含多个子Object3D

        Object.defineProperties(this, {
            // 局部位置(平移。初始位置:(0,0,0))
            position: {
                configurable: true,
                enumerable: true,
                value: new Vector3()
            },
            // 局部旋转
            // Quaternions are used in three.js to represent rotations
            quaternion: {
                configurable: true,
                enumerable: true,
                value: new Quaternion()
            },
            // 局部缩放
            scale: {
                configurable: true,
                enumerable: true,
                value: new Vector3(1, 1, 1)
            },
            // modelViewMatrix
            // 等于 摄像机的世界模型矩阵的逆矩阵 乘以 物体的世界模型矩阵，详见Matrix4中的解释
            modelViewMatrix: {
                value: new Matrix4()
            },
            // 用于变换顶点法向量的”魔法矩阵“，modelViewMatrix的逆转置矩阵。参考“《WebGL编程指南》摘要.md”
            normalMatrix: {
                value: new Matrix3()
            }
        });

        // 局部模型矩阵
        // 包含了this.position, this.quaternion, this.scale信息，即位置(平移)、旋转、缩放
        this.matrix = new Matrix4();
        // 世界模型矩阵
        this.matrixWorld = new Matrix4();

        // ...
    }

    // 根据this.position, this.quaternion, this.scale更新this.matrix
    updateMatrix() {
        // 由this.position, this.quaternion, this.scale生成this.matrix
        this.matrix.compose(this.position, this.quaternion, this.scale);
        this.matrixWorldNeedsUpdate = true;
    }
    // 根据this.matrix更新this.matrixWorld，并更新所有child的matrixWorld
    updateMatrixWorld(force) { }

    // 局部旋转
    rotateOnAxis(axis, angle) { }
    // 局部平移
    translateOnAxis(axis, distance) { }
    // 通过一个矩阵来进行局部旋转、局部平移、局部缩放
    applyMatrix4(matrix) {
        this.matrix.premultiply(matrix);
        // 将this.matrix中的位置、旋转、缩放分量分别提取this.position, this.quaternion, this.scale中
        this.matrix.decompose(this.position, this.quaternion, this.scale);
    }

    // 增删子Object3D
    add(object) { }
    remove(object) { }

    // ...
}

Object3D的平移、旋转、缩放方法并不会去修改顶点的位置坐标(Object3D及其子类对象并不存储顶点的位置坐标)，而是通过间接的修改局部模型矩阵matrix来实现平移、旋转、缩放。

对比：BufferGeometry及其子类对象存储了每一个顶点的位置坐标，对它们的平移、旋转、缩放是通过修改顶点的位置坐标来实现的。

    rotateZ(angle) { }
    translate(x, y, z) { }
    scale(x, y, z) { }

    // 设置顶点属性(this.attributes[name])

        this.attributes[ name ] = attribute;
        return this;

    // 设置顶点坐标(this.attributes['position'])
    setFromPoints(points) {

        for ( let i = 0, l = points.length; i < l; i++ ) {
            const point = points[ i ];
            position.push( point.x, point.y, point.z || 0 );
        }
        this.setAttribute( 'position', new Float32BufferAttribute( position, 3 ) );
    }
